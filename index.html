FARMING FRENZY

C#

Game Development

Farming Frenzy is a game I created in Unity to demonstrate game design concepts. The gameplay loop revolves around careful resource management, with the aim of maintaining a "happiness" value on your farm by ensuring that the animals are well looked after and crops are being grown in suitable time. Across five levels, the challenge is increased by introducing new tasks on the farm, and increasing the distance between tasks to increase the player's cognitive load.

A demonstration video is available here.

MINITRIANGLE COMPILER

Haskell

I produced a compiler for the "minitriangle" language. The compiler took .MT code and converted it to an abstract machine code implementation that ran in a simulated environment. I then further extended the compiler to increase the instruction set, and add basic types.
